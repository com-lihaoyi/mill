== `jlink` and `jpackage`

Mill has builtin support for the https://docs.oracle.com/en/java/javase/11/tools/jlink.html[JLink] and
https://docs.oracle.com/en/java/javase/17/docs/specs/man/jpackage.html[JPackage] command line tools.
For more details, see:

* xref:javalib/package.adoc#_java_app_and_bundles_using_jlink[Java App and Bundles using JLink]
* xref:javalib/package.adoc#_java_installers_using_jpackage[Java Installers using JPackage]
